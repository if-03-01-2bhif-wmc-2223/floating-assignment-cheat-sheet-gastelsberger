Advanced CSS

Box Model

A block element starts on a new line and fills up space left/right of the webpage.

  • Margins extend the "box" on the outside
  • Paddings extend the "box" on the inside

Margins and paddings can be set for the top/bottom and for left/right.

Inline elements do not start on a new line and margins/paddings can only be set on the left/right. Inline-blocks provide an exception to that.

Classes and IDs

Classes can be used n-times, IDs can only be used once per page. IDs select one element, classes select one or more elements.

Classes are selected by using .(CLASSNAME), HTML elements can be selected using (HTMLELEMENT).(CLASSNAME).

IDs are selected with #ID. HTML elements can be selected using (HTMLELEMENT)#ID.

Floating

A HTML element can float if you want it to. Floating is achieved by: ELEMENT {float: none/left/right;}
