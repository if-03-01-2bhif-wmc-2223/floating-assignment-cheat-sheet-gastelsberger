CSS

CSS stands for Cascading Style Sheets.
Every css rule has a selector, a value, a property and a declaration.

h1 // SELECTOR
{
color (property): blue (value);
}

WARNING!
The div element is only a placeholder in the following cases.

Direct child: Selects all elements that are children of a specified element. (Selects all p's that are children of a div element)
div > p {color: blue;}

Descendant: Selects all elements that are children of a specific element. (Selects all p's inside div elements)
div p {color: blue;}

Adjacent sibling: Selects the first p element of a div element.
div + p {color: blue;}

Sibling: Selects all elements that are siblings of a specific element. (Let's say you have 4 paragraphs. Two within a div tag, two not in a div tag. The two statements not within the div tag will have the color blue.)
div ~ p {color: blue;}

Colors in a hexadecimal notation:
(opening) h1 style="color:#ff3813;" (closing) h1

Declaration for background image:
body{
background-image: url("yes.png");
}

Text alignation (left, right, center, justify):
h1{
text-align: left;
}

Text decoration (overline, underline, line-through):
h1{
text-decoration-line: overline;
}

Text transformation (uppercase, lowercase, capitalize):
h1{
text-transform: uppercase;
}

Font family declaration:
p.a{
font-family: "serif";
}

Font style declaration:
p.a{
font-style: "normal";
}

Font size declaration:
p.a{
font-size: 20px;
}

Font weight declaration:
p.normal{
font-weight: normal;
}
